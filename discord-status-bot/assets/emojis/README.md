# Custom Emoji Assets

This folder is for your custom Discord emoji images.

## How to use:

1. **Add your emoji images here:**
   - `green.webp` - for online services
   - `yellow.webp` - for slow services  
   - `red.webp` - for offline services

2. **Upload to Discord:**
   - Go to your Discord server settings
   - Navigate to "Emoji" section
   - Upload your images and note the emoji IDs

3. **Update the bot:**
   - Run `npm run update-emojis` to update the emoji IDs in the bot

## File naming convention:
- Use descriptive names like `green.webp`, `yellow.webp`, `red.webp`
- Supported formats: PNG, JPG, GIF, WEBP
- Recommended size: 32x32px or 64x64px for best quality
