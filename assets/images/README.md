# Example Images

Place your example profile screenshot here.

**Supported formats:** `.png`, `.jpg`, `.jpeg`

**File naming:**
- Name your file: `example-profile.png` (or `.jpg`/`.jpeg`)
- The bot will automatically use this image in the verification channel instructions

**How to add:**
1. Take a screenshot of an example 8 Ball Pool profile
2. Save it as `example-profile.png` in this folder
3. Restart the bot or use `!instructions` command

The bot will attach this image to the instructions message in the verification channel.
