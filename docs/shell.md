# Shell Utilities

All shell utilities live in `shell/`. Backwards-compatible symlinks remain at the original paths.

- deploy.sh – Deploy full system
- docker-deploy.sh – Build/deploy via Docker Compose
- fix-screenshot-permissions.sh – Reset permissions under screenshots/
- migrate-to-docker.sh – Migrate processes to Docker
- rename-project.sh – Rename project safely
- setup-cloudflare-tunnel.sh – Configure Cloudflare Tunnel
- setup-mongodb.sh – Provision MongoDB
- vps-setup.sh – Initial VPS provisioning
- shell/check-port-conflicts.sh – Check for port collisions
